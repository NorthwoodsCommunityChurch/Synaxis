# Reference Documentation

Downloaded January 29, 2026. These are the primary protocol and API references for Touchdrive to Premiere.

## Files

### HyperDeckEthernetProtocol.pdf
- **Source:** Blackmagic Design (December 2024)
- **URL:** https://documents.blackmagicdesign.com/DeveloperManuals/HyperDeckEthernetProtocol.pdf
- **Contents:** Complete HyperDeck Ethernet Protocol reference — TCP port 9993, all commands, response codes, async notifications, transport states, clip management, timecode configuration

### RESTAPIForHyperDeck.pdf
- **Source:** Blackmagic Design (December 2024)
- **URL:** https://documents.blackmagicdesign.com/DeveloperManuals/RESTAPIForHyperDeck.pdf
- **Contents:** REST API reference for HyperDeck Extreme models — HTTP endpoints for transport, system, media, timeline control. Includes WebSocket event subscription endpoint.

### RossTalk-Commands.pdf
- **Source:** Ross Video (v32)
- **URL:** https://documentation.rossvideo.com/files/Manuals/Production%20Switchers/Protocol%20Documents/RossTalk-Commands(4802DR-403).pdf
- **Contents:** Complete RossTalk command reference for Carbonite, Graphite, Acuity, Vision, Ultrix, and XPression. TCP port 7788, ASCII protocol, all command syntax.

### ProPresenter-Official-OpenAPI.json
- **Source:** Renewed Vision (official swagger.json from openapi.propresenter.com)
- **URL:** https://openapi.propresenter.com/swagger.json
- **Contents:** Official OpenAPI/Swagger specification for the ProPresenter REST API. Introduced in ProPresenter 7.9. All endpoints, request/response schemas, parameter definitions.

### ProPresenter-OpenAPI-Spec.json
- **Source:** Jeff Mikels (community mirror of the 7.9 spec)
- **URL:** https://github.com/jeffmikels/ProPresenter-API/blob/master/pro7.9.openapi-spec.json
- **Contents:** Community-maintained copy of the ProPresenter 7.9 OpenAPI spec with additional documentation notes.
